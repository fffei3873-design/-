☁️ Supabase 云端配置指引
1

注册 Supabase 账号

访问 supabase.com，免费注册账号并创建一个新项目（选择离你最近的区域，如 Singapore）

2

创建数据表

进入项目 → SQL Editor，粘贴并运行以下 SQL：

CREATE TABLE todos ( id uuid DEFAULT gen_random_uuid() PRIMARY KEY, date text NOT NULL, time text DEFAULT '', cat jsonb DEFAULT '[]', task text NOT NULL, note text DEFAULT '', done boolean DEFAULT false, carried boolean DEFAULT false, carried_from text DEFAULT '', recurring boolean DEFAULT false, recurring_rule text DEFAULT '', recurring_parent_id text DEFAULT '', recurring_expanded boolean DEFAULT false, carry_count int DEFAULT 0, source text DEFAULT '', created_at timestamptz DEFAULT now(), updated_at timestamptz DEFAULT now() ); -- 允许匿名读写（个人使用） ALTER TABLE todos ENABLE ROW LEVEL SECURITY; CREATE POLICY "allow_all" ON todos FOR ALL USING (true) WITH CHECK (true);
3

获取连接信息

进入 Settings → API，复制：
· Project URL（形如 https://xxx.supabase.co）
· anon public key（以 eyJ 开头的长字符串）

4

填入工具设置

点击右上角设置图标 → 云端数据库，粘贴 URL 和 Key，保存连接即可
